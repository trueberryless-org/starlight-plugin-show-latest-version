Das Starlight Cooler Credit Plugin kann in der Konfigurationsdatei `astro.config.mjs` deines Projekts konfiguriert werden:

```js {11}
// astro.config.mjs
import starlight from "@astrojs/starlight";
import { defineConfig } from "astro/config";
import starlightCoolerCredit from "starlight-view-modes";

export default defineConfig({
    integrations: [
        starlight({
            plugins: [
                starlightCoolerCredit({
                    // Die Konfigurationsoptionen findest du hier.
                }),
            ],
            title: "My Docs",
        }),
    ],
});
```

## Konfigurationsoptionen

Das Starlight Cooler Credit Plugin akzeptiert die folgenden Konfigurationsoptionen:

### `credit`

**Typ:**

```ts
"Astro" |
"Starlight" |
{
    title: string | Record<string, string>;
    href: string;
    description?: string | Record<string, string> | undefined;
}
```

**Standard:** `"Starlight"`

Mit der Option `credit` kannst du den Anerkennungstext konfigurieren, der im Inhaltsverzeichnis der Starlight-Website angezeigt wird. Es gibt [drei vorkonfigurierte Optionen](#vorkonfigurierte-optionen): `"Astro"`, `"Starlight"` und `"Starlight Blog"`. Du kannst auch ein benutzerdefiniertes Objekt angeben, um den Anerkennungstext für jedes Gebietsschema zu konfigurieren.

:::note
Bitte beachte, dass du mindestens den Gebietsschema-Wert deiner [`defaultLocale`](https://starlight.astro.build/de/reference/configuration/#defaultlocale) konfiguriert haben musst, wenn du Anerkennungstext für jedes Gebietsschema bereitstellen willst.
:::

Hier ist ein Beispiel für eine benutzerdefinierte Konfiguration:

```ts
starlightCoolerCredit({
    credit: {
        title: {
            en: "Built with Astro",
            de: "Erstellt mit Astro",
        },
        href: "https://docs.astro.build/",
        description: {
            en: "Want to build your own static website?",
            de: "Willst du deine eigene statische Website erstellen?",
        },
    },
}),
```
